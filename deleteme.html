In this module, we will start by briefly discussing some of the new HTML5 elements and review images and how to validate our HTML documents. This will lead us right into the basics of Cascading Style Sheets. This module contains a few pages on the CSS Box Model; using colors with HTML, and formatting text on a page. 

Module Objectives

  • Create a web page using a basic HTML5 page structure with HTML5 elements

  • Learn more about external and internal  CSS stylesheets 

  • Use W3C online validation service to validate HTML pages

  • Describe the most common CSS selectors

  • Utilize id and class attributes

  • Demonstrate using colors for text and background on a web page.
